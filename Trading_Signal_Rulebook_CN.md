# 交易信号机器人规则说明书（中文版）

日期：2026-06-09  
用途：供与股票专家讨论当前规则框架、参数与风险控制逻辑

## 1. 系统定位

- 本系统是“规则驱动的技术信号提醒工具”，用于辅助你做主观交易决策。
- 系统不自动下单，不连接自动交易执行流程。
- 运行方式：
  - 持续监控：`python run.py`
  - 主动问答：`python ask.py price SYMBOL` / `python ask.py advice SYMBOL`

## 2. 当前监控范围与周期

- 当前监控清单：
  - `HK.07709`
  - `US.SPY, US.QQQ, US.MU, US.SNDK, US.GLW, US.IREN, US.TSLA, US.MSFT, US.NVDA`
- 多周期结构：
  - 执行周期：`15m`
  - 趋势过滤周期：`1h`
  - 确认周期：`5m`、`1m`

## 3. 指标计算项

每个标的、每轮轮询都计算：

- `MA20`、`MA50`
- `RSI(14)`
- 最近摆动高点 / 摆动低点（回看 20 根）
- 成交量比（当前量 / 20 均量）
- `ATR(14)` 与 `ATR% = ATR / 收盘价`

数据最低要求：

- 至少 60 根 K 线才能完成全套指标计算。

## 4. 信号类型与优先级

系统可能输出 4 类信号：

- `exit_warning`（风控/减仓提醒）
- `breakout_candidate`（突破候选）
- `long_setup`（偏多进场候选）
- `observation_only`（观察）

同一时刻只返回 1 类，优先级顺序为：

1. `exit_warning`
2. `breakout_candidate`
3. `long_setup`
4. `observation_only`

## 5. 信号规则细则

### 5.1 `long_setup`（偏多候选）

需要同时满足：

- 15m：价格高于 `MA20` 且高于 `MA50`
- 15m：`MA20 > MA50` 或 `MA20` 在上升
- RSI 恢复：上一根 RSI < 48，当前 RSI >= 48
- 1h 趋势支持：1h 价格 > 1h `MA20` 且 1h `MA20 >= MA50`
- 距离阻力不过近：到摆动高点距离 > 0.8%
- 低周期确认：`5m/1m` 至少 1 个满足（价格 >= MA20 且 RSI >= 50）
- ATR 过滤通过（开启时）
- 相对强弱过滤通过（开启时）

输出包含：

- 入场参考区间（当前价上下 0.4%）
- 失效位（`min(MA20, swing_low)`）
- 最近阻力位（`swing_high`）

### 5.2 `breakout_candidate`（突破候选）

需要同时满足：

- 价格上破最近摆动高点
- 成交量比 >= 1.2
- 1h 趋势不弱（1h 价格 >= 1h MA20）
- 不过度拉伸（突破后涨幅 <= 3.0%）
- 低周期结构未破（确认周期中至少一个价格 >= MA20）
- ATR 过滤通过（开启时）
- 相对强弱过滤通过（开启时）
- 突破结构通过（开启时）：
  - 新鲜突破（前收未破、当前破位），或
  - 控制性回踩后站回（容差 0.4%）

输出包含：

- 入场区间（突破位到当前价）
- 失效位（近 MA20）
- 动态阻力（当前价 * 1.03）

### 5.3 `exit_warning`（风控提醒）

满足任一条件即触发：

- 价格跌破 MA20 且 RSI 继续走弱
- RSI < 42
- 价格跌破摆动低点（支撑失守）

该信号风险等级固定为 `high`。

### 5.4 `observation_only`（观察）

满足以下条件时触发：

- 市场有改善迹象（价格 > MA20 或 RSI 上升）
- 1h 结构尚未明显转坏（1h 价格 >= 1h MA50）

含义：有修复但未充分确认，建议继续观察。

## 6. 质量分与优先级分层

- 信号会计算 `quality_score`（0-100）
- 证据强度：
  - `strong`：>= 80
  - `medium`：>= 60
  - `weak`：< 60
- 优先级：
  - `A`：>= 80
  - `B`：>= 60
  - `C`：< 60

## 7. 过滤与提醒抑制机制

### 7.1 ATR 波动过滤

- 当前开启：`true`
- 合法区间：`0.3% <= ATR% <= 5.0%`

### 7.2 相对强弱过滤

- 当前设置：关闭（`ENABLE_RELATIVE_STRENGTH_FILTER=false`）
- 开启后逻辑：标的回看收益需 >= 基准收益 + 超额阈值

### 7.3 冷却 + 去重

- 同标的同信号类型冷却：30 分钟
- 若与上次同类型信号价格偏差 < 0.3%，抑制重复提醒

### 7.4 最低推送等级

- 当前为 `C`（A/B/C 全部可推送）
- 若调整为 `B` 或 `A`，低等级会记录但不推送

### 7.5 同标的信号合并提醒

- 当前开启：`true`
- 窗口：20 分钟
- 若窗口内信号类型变化，发送“合并更新提醒”而不是重复刷屏

## 9. 报表机制

- 日报：已开启，16:10（美东时区），Top 5
- 周报：已开启，周五 16:20（配置 weekday=4），Top 8

报表内容包括：

- 信号总数
- 平均质量分
- 按类型/等级分布
- 活跃标的排行

## 10. 当前关键参数（供讨论）

- `RSI_RECOVERY_THRESHOLD = 48`
- `EXIT_RSI_THRESHOLD = 42`
- `VOLUME_RATIO_MIN = 1.2`
- `RESISTANCE_BUFFER_PCT = 0.008`
- `MAX_EXTENSION_PCT = 0.03`
- `ENTRY_ZONE_PCT = 0.004`
- `ALERT_COOLDOWN_MINUTES = 30`
- `DUPLICATE_PRICE_TOLERANCE_PCT = 0.003`
- `ALERT_MIN_PRIORITY_TIER = C`
- `POLL_INTERVAL_SECONDS = 60`

建议与专家重点讨论：

- 入场/退出 RSI 阈值是否适配你的风格
- 突破量能阈值 1.2 是否需要动态化
- 3% 拉伸上限在不同波动阶段是否过紧/过松
- 是否将最小推送等级提升到 `B` 降噪
- 是否开启相对强弱过滤用于盘中筛选
